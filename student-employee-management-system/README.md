# Student Employee Management System

## Overview

The Student Employee Management System is a backend application built using Java and Spring Boot to manage student and employee records.

This project is a modern rebuild of my first software project developed during my first year of B.Tech. It follows industry-standard backend development practices and will evolve into a production-quality application.

## Planned Features

- Student Management
- Employee Management
- REST APIs
- Authentication & Authorization (JWT)
- Validation
- Exception Handling
- PostgreSQL
- Docker
- Unit & Integration Testing

## Tech Stack

- Java
- Spring Boot
- Spring Security
- PostgreSQL
- Maven
- Docker
